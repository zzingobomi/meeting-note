# Dev Https 실행

- desktop, notebook 둘다 lovalhost 일때는 정상적으로 작동
- getUserMedia 는 https or localhost 에서만 실행 가능
- set HTTPS=true&&npm start (HTTPS 로 실행)
- set HTTPS=false&&npm start (HTTP 로 실행)
- server.js origin domain 에 맞게
